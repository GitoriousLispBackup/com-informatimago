Sudoku Solver

This file contains a sudoku solver.

  • sudoku-solver.lisp

Example: 

cl-user> (use-package :com.informatimago.sudoku-solver)
t
cl-user> (let* ((sudoku #2A((x x x 8 x 4 2 x x)
                            (6 x 8 x 2 x x x 4)
                            (2 1 x 6 5 3 x x 8)
                            (x 7 x 2 x 6 x 9 x)
                            (x x x x 3 x 1 x x)
                            (4 2 3 x x 9 x 5 7)
                            (x 6 x 4 1 5 7 x x)
                            (x x 7 x x 8 3 x x)
                            (x 5 9 x x x x 1 x)))
                (solutions (sudoku-solver sudoku)))
           (terpri)
           (sudoku-print sudoku)
           (format t " has ~D solution~:*~P.~2%" (length solutions))
           (map nil 'sudoku-print solutions))
